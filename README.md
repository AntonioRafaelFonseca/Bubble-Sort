# Bubble Sort Visualization with p5.js

This project is a visualization of the **Bubble Sort algorithm** using [p5.js](https://p5js.org/).

It's possible to run  [**sketch.js**](https://github.com/AntonioRafaelFonseca/Bubble-Sort/blob/master/BubbleSort/sketch.js) in your browser at [p5.js web editor](https://editor.p5js.org/)
however if you want to run it in a Code Editor you need all the files
